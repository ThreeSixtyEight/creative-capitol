## Prereq
Python  
pip  

## Upgrade pip
`pip install --upgrade pip`

## Install packages
`pip install opencv-python`  
`pip install mediapipe`  
`pip install flask`  
`pip install flask_socketio`  
`pip install flask_cors`  
`pip install eventlet`  
## Run code
`python3 socket_server_display.py`  
`python3 socket_server.py`
